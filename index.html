1. PROXY CALETAS.

Un proxy es la manera más fácil de acceder al contenido bloqueado.

Imagínate que entre tú y el contenido hay una montaña. El proxy está en el tope de esa montaña y es capaz de ver hacia el otro lado. Tu le gritas preguntándole lo que quieres ver y él, como tiene vista directa, te responde con lo que quieres saber.

Instrucciones

  • Entra en el Proxy de Caletas y escribe la dirección de la página que quieres acceder

2. DNS.

Los DNS "traducen" la dirección pronunciable de una página –como www.google.com– a la dirección real del servidor que es en números (por ejemplo: 173.194.70.113).

Si el DNS no hace bien la traducción, no podrás llegar a la página deseada.

Para asegurarte que eso no pase, usa uno de estos dos DNSs confiables:

Google DNS

  • Instrucciones para configurar Google DNS en Windows 7 y en MacOS
  • Guía y explicación oficial de Google en inglés.

OpenDNS

  • Instrucciones para OpenDNS en Windows 7 y en MacOS
  • Página Oficial de OpenDNS

2. VPN.

Un VPN es una especie de túnel seguro que permite acceder a una red remota – en este caso la red remota es Internet

Ese túnel cifra tu información y te hace ver como si estuvieras conectando desde otra locación física.

F-Secure - Freedome

  • En nuestra opinión, el mejor VPN. Tiene apps para todas las plataformas, la interfaz y experiencia de usuario son excelentes y, sobre todo, muy segura: Freedome
  • Es paga pero la historia nos ha demostrado que no se puede confiar en VPN gratuitos

ProXPN

  • Tienen una versión gratuita (no muy rápida y sólo para acceder al web, pero muy segura y efectiva)
  • Bájalo para Mac o Windows

4. TOR

Este es el peso pesado, debería funcionar en caso que todo lo anterior falle.

Tor es un software gratis (y una red abierta) que permite usar Internet anónimamente* y sin revelar tu ubicación física.

  • Bajen el browser que viene preconfigurado y listo para usarlo. Hay versiones para MacOS, Windows y Linux:
  • Para Android, pueden bajar Orbot

* en teoría te permite usar la red anónimamente pero para eso se tienen que tomar otras previsiones más técnicas. Nuestra recomendación es usarlo para acceder a páginas bloqueadas pero asuman que la comunicación no es completamente anónima.

5. Mensajeros

Whatsapp*, Telegram*, Viber, Line y los SMS no son seguros.

Te recomendamos dos opciones para móviles y una para Web:

Signal

  • Signal, creada por Open Whisper Systems, es por rato largo una de las mejores apps para comunicarte de manera segura, te permite hacer llamadas y mandar mensajes en iOS y Android.

Telegram* (esta es la media opción)

  • Agrego Telegram a la lista porque hay mucha gente que lo tiene instalado y la seguridad es mucho mejor que la de Whatsap, sms, etc.
  • Lo puedes bajar para Android y para iOS

Crypto Cat

  • Crypto Cat es un programa de mensajería segura que funciona desde el browser
  • Lo puedes bajar aquí
